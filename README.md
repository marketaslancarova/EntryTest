# Tiles – Interview Demo Project

- **Backend** – Node.js + Express (REST API for working with `tiles`)
- **Frontend** – Angular 17 (standalone components with signals)

## Requirements

- Node.js (version ≥ 18 recommended)  
- npm (comes with Node.js)  
- Git (optional)
